Downloading and Installing Cygwin

The easiest way to install Cygwin is to simply click on the "Install Cygwin Now!" link on the Cygwin home page. This will download the setup.exe program to your computer. When you run setup.exe you will be asked a series of questions for installing Cygwin on your computer. We highly recommend this method for installing Cygwin.

The setup.exe program allows you to download the Cygwin distribution to your computer without installing it. By copying the download directory to other systems or putting it on a file server, you can, at a later time, use setup.exe to install directly from your hard disk.

Due to volume limitations, the Cygwin distribution is only available from mirror sites. You'll be prompted to choose one of these during the setup.exe installation.

If you are adventurous (i.e., masochistic) you can also download the Cygwin distribution via ftp or http. Currently, the distribution is located in two directories: "latest" and "contrib". That is subject to change at any time, though.

The Cygwin distribution consists of standard UNIX .tar.gz or .tar.bz2 files. Cygwin's tar + gzip + bzip2 programs can unpack these files, just like their UNIX counterparts however you will probably find yourself in a chicken/egg situation if you attempt this without Cygwin installed on your system. Installing Cygwin using this method is not recommended.

Note that you may be tempted to unpack the .tar.gz files with some Windows program like WinZip. This will not work reliably. Cygwin's tar files are fully understood by WinZip. Attempting to use WinZip to unpack a distribution will likely result in a damaged distribution.

If you don't want to use the setup program, there is an additional file called bootstrap.zip that has a few key programs you'll need to unpack the tar.gz files. If you need further help doing it this way, you probably should be using the setup program:-)
