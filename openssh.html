Cygwin's OpenSSH is the Cygwin port of that famous implementation of the SSH/SecSH protocol. And the best of all, it's FREE. Cygwin's OpenSSH runs on Windows 95/98/ME/NT and 2000. Even the server part (sshd) runs on all these Windows versions.

The purpose of this page is to help people installing OpenSSH in their Windows environment. However, the page is still under construction. So you can only read the Cygwin OpenSSH README here.

Notes
The current version of OpenSSH is 2.5.1. the Cygwin version is based on the original sources of the portable OpenSSH version 2.5.1p2. All Cygwin related changes are part of the original source tree.

You can get binaries and sources of Cygwin's OpenSSH by clicking on the link Install Cygwin now to the right.

You can get the sources also from the official mirrors of the portable OpenSSH or from the official CVS repository.

If you want to contribute to this web page, feel free to contact us by sending mail to the Cygwin mailing list. We would be glad about interesting articles about using and configuring Cygwin's OpenSSH as well as to hear of new and interesting books or links related to security, the Secure Shell Protocol and it's relation to Cygwin/Windows.
