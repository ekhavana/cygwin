Is there a mailing list for the project?

Yes. There are several:

  • cygwin: a list for discussion of all things Cygwin-related. If you have questions about how to use Cygwin, or any of its tools (bash, gcc, make, etc.), this is the list for you. HTML mail is NOT PERMITTED on this list and will be stripped.
    (traffic usually consists of 15-40 messages/day)

  • cygwin-announce: a moderated announcement list for those folks who want to avoid all the traffic.
    Note that announcements sent to cygwin-anounce are automatically sent to the cygwin mailing list so there is no reason to subscribe to both mailing lists.

  • cygwin-developers: a by-approval developers list for discussing the internals of the Cygwin DLL and the other components of the "winsup" directory. Discussions of upcoming net releases are also appropriate. Subscription requests are handled on a case-by-case basis. Only subscribers may submit email to this list. Please do not join the cygwin-developers mailing list unless 1) you are thoroughly familiar with the cygwin DLL, 2) you have successfully built the DLL from source, and 3) you intend to contribute source code to Cygwin.

  • cygwin-patches: a by-approval developers list for submitting patches to the Cygwin DLL and the other components of the "winsup" directory. Discussions of supplied patches are also acceptable, of course. Only subscribers may submit email to this list.

  • cygwin-apps: a by-approval developers list for discussing issues regarding applications that use the Cygwin DLL. If you are maintaining one of the packages that is distributed with the Cygwin net releases you should be subscribed to this list. Only subscribers may submit email to this list.

  • cygwin-xfree: a list for discussion of porting Xfree86 to Cygwin.

  • cygwin-cvs: a list which receives log messages for every CVS commit in the 'winsup' directory. If you don't know what this means then you probably shouldn't subscribe. Messages consist of a log message and pointers to diffs of the changes.

You can subscribe to or unsubscribe from any of the above lists by sending email to themailinglistname-subscribe@sourceware.cygnus.com or themailinglistname-unsubscribe@sourceware.cygus.com respectively.

If you have any questions or problems with the mailing lists, see the Sourceware mailing list FAQs.

Before you send any mail to one of these lists, you need to read the relevant Cygwin documentation, especially the Cygwin FAQ, and the mailing list archives so that you don't ask a question that someone has already asked or report a bug that has aleady been reported.

We try to actively reduce the amount of spam sent to the lists by using the MAPS and ORBS relay blocking lists. If your mail server relays mail, note that it may be listed in one of these servers and you will not be able to send mail to sourceware.cygnus.com. Should that be the case, contact your mail admin about fixing your mail system.

Questions or problems with the mailing lists should be directed to postmaster@sourceware.cygnus.com.
