Is there a mailing list for the project?

Yes. In fact there are three:

  • gnu-win32: a list for discussion of all things Cygwin-related (traffic usually consists of 15-40 messages/day)
  • gnu-win32-announce: a moderated announcement list for those folks who want to avoid all the traffic. (Note that mail sent to the moderated list is also sent to the discussion list so you should only need to be subscribed to one).
  • cygwin32-developers: a developers list for discussing the internals of the Cygwin library. Please only join if you intend to contribute code to Cygwin.

To subscribe to the main list, send a message to gnu-win32-request@cygnus.com with a message body of

subscribe gnu-win32

To join the announcement list, send a message to gnu-win32-announce-request@cygnus.com with a message body of

subscribe gnu-win32-announce

The same method can be used to join the cygwin32-developers list.

To get more information about the mailing lists or the list server (including info on how to unsubscribe from either list), send a message to "gnu-win32-request@cygnus.com" with one line of text: "help". Note that following a successful unsubscribe, you may still receive mail from the list for a day or two because it can often take that long for the mail queue to empty.

Before you send any mail to the list, you need to read the relevant Cygwin documentation, especially the Cygwin FAQ, and the mailing list archives so that you don't ask a question that someone has already asked or report a bug that has aleady been reported.
