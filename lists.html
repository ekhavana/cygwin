Is there a mailing list for the project?

Yes. In fact there are four:

  • cygwin: a list for discussion of all things Cygwin-related (traffic usually consists of 15-40 messages/day)
  • cygwin-xfree: a list for discussion of porting Xfree86 to Cygwin.
  • cygwin-announce: a moderated announcement list for those folks who want to avoid all the traffic. (Note that mail sent to the moderated list is also sent to the discussion list so you should only need to be subscribed to one).
  • cygwin-developers: a by-approval developers list for discussing the internals of the Cygwin DLL and the other components of the "winsup" directory. Discussions of upcoming net releases are also appropriate.

    Please only join if you intend to contribute code to Cygwin. Subscription requests are handled on a case-by-case basis. Only subscribers may submit email to this list.

You can subscribe to or unsubscribe from any of these lists by using the form below. cygwin-developers is a by-approval only list; even though you send in a subscription request, it may not be approved.

If you have any questions or problems with the mailing lists, see the Sourceware mailing list FAQs.

Before you send any mail to one of these lists, you need to read the relevant Cygwin documentation, especially the Cygwin FAQ, and the mailing list archives so that you don't ask a question that someone has already asked or report a bug that has aleady been reported.

We try to actively reduce the amount of spam sent to the lists by using the MAPS and ORBS relay blocking lists. If your mail server relays mail, note that it may be listed in one of these servers and you will not be able to send mail to sourceware.cygnus.com. Should that be the case, contact your mail admin about fixing your mail system.

Questions or problems with the mailing lists should be directed to postmaster@sourceware.cygnus.com.
