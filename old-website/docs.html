Cygwin Documentation

  • Cygwin User's Guide
  • Cygwin API Reference
  • Frequently Asked Questions (FAQ) file for the current Cygwin beta release. Here's a version of the FAQ translated into Japanese.
  • Cygwin: A Free Win32 Porting Layer for UNIX Applications. On August 3, 1998, Geoffrey Noer presented this paper at the 1998 Usenix NT Symposium. It provides an overview of the project and goes into the implementation of Cygwin in some detail, although much of it may now be dated. Available in HTML, Word 98, and Postscript formats.

Some work remains to be done in documenting the Cygwin API. In addition to the Cygwin API Reference and information in the FAQ, you may want to check source code file winsup/cygwin.din which lists all of the Cygwin DLL exports.

We have some preliminary docs on how to profile the Cygwin DLL.

Also note that there is quite a lot of useful information in the project mailing list archives.

Stabs

The tools use a debugging info format called stabs. You can also find this document in GDB (GNU Debugger) distributions. Other preformatted versions (.txt, PostScript) are available from the ftp location.
