I'm still running Windows 95/98/Me, no Cygwin for me?

Windows 95, Windows 98, and Windows Me are really old and quite crappy Windows versions which are missing important functionality and are just not up to the times anymore. With the introduction of Windows XP a lot of projects stopped to support these versions long ago. Beginning with Cygwin 1.7 we are performing this step as well and concentrate on NT-based Windows versions.

However, as a curtesy to the occasional user of one of these systems, we still provide the old 1.5.25 release of Cygwin. Here's how you get it:

To install Cygwin 1.5.25, download the legacy installer application setup-legacy.exe and run it.

The legacy documentation for Cygwin 1.5.25 is still available for reference. There's the old User's Guide, the old FAQ, and the old API reference.

Please keep in mind that this legacy version of Cygwin is not supported any longer. If you're running Windows 95, 98 or Me, there's no problem to ask questions on the appropriate Cygwin mailing list. If you running any other version of Windows, be prepared to be redirected to the latest Cygwin release.

Just in case this information has been missed: The legacy Cygwin release 1.5.25 is only kept available for the sake of Windows 95/98/Me users. It's not supported anymore and if you're using any other version of Windows, install the latest Cygwin instead.
