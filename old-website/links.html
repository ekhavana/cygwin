  • GNU-Win32 extras can be found here, including Cygwin32 native compiler binaries based on gcc sources from the EGCS project. Mingw32:

Mingw32 ("Minimalist GNU-Win32") is a gcc compiler configuration that does without the Cygwin32 Unix emulation layer. Instead, it uses the Microsoft CRTDLL as its C library. If you want a Win32 native compiler without Unix support, this is for you! Eventually Cygwin32 and Mingw32 tools will be combined into a single configuration called ix86-win32 that will allow the user to choose whether or not to use the Unix emulation or not.

  • The original Mingw32 site by Colin Peters who started Mingw
  • Jan Jaap's site containing gcc 2.8 binaries for Mingw and other related resources
  • Mumit Khan's i386-mingw32 native toolchain based on egcs is available here.
