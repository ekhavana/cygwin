What are the licensing terms?

Most of the tools are covered by the GNU GPL, some are public domain, and others have a Berkeley style copyright. To cover the GNU GPL `restrictions', the basic rule is if you give out any binaries, you must also make the source available. For the full details, be sure to read the text of the GNU General Public License (GPL).

The Cygwin32 API library found in the winsup subdirectory of the source code is also covered by the GNU GPL. By default, all executables link against this library (and in the process include GPL'd Cygwin32 glue code). This means that unless you modify the tools so that compiled executables do not make use of the Cygwin32 library, your compiled programs will also have to be free software distributed under the GPL with source code available to all.

Cygnus' Native Win32 GNUPro product includes a commercial license for Cygwin32 that is more suitable for commercial use of the Cygwin32 library. Contact info@cygnus.com for more information about this license.
