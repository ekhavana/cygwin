 Data Plotting Library DISLIN 7.3
DISLIN is a high-level and easy to use plotting library for displaying data as curves, bar graphs, pie charts, 3D-colour plots, surfaces, contours and maps. Several output formats are supported such as X11, VGA, PostScript, CGM, HPGL, TIFF, PNG and Prescribe. The library contains about 400 plotting and parameter setting routines and is available for several C, Fortran 77 and Fortran 90 compilers. Mar 11 2000, Helmut Michels
