Cygwin GNOME 1.4 binary packages - Dec 10 2002, Hansom Young
I've setup a project on sourceforge.net named cygnome, aiming to port GNOME desktop to Cygwin. Currently most of GNOME 1.4 packages are available, including core libraries, the GNOME desktop, Sawfish window manager, gnumeric and anjuta. But Nautilus is not available now. Most of these are built with the patches and hints contributed by Steven O'Brien at http://homepage.ntlworld.com/steven.obrien2. With his great work, this port come to reality. I'm providing these binary packages just in hope that it could be useful, esp. for those guys who have less time to build it by themselves.

Unofficial Off-site Cygwin News Server - Mar 2 2002, Daniel Adams
A news server for the cygwin mailing list has been available for a little while now. it is available at news://news.gmane.org/gmane.os.cygwin To me and many others this is a great rescource. While I am not the person who may have started the news server, I am an advocate of it. This server is run by an external service and is not officially associated with the Cygwin project. Problems or questions should go to the gmane.org site not the cygwin mailing list.

Franken Archives Moved - Apr 19 2001, Michael Hirmke
The archives at franken.de have been moved to ftp.uni-erlangen.de
to support the increase in download traffic from the Cygwin areas.
ftp.uni-erlangen.de has a 155Mbit connection so the change should
be a good thing for all users. Please check all your links and
make sure they reflect the new name. The upload of ported packages
is still allowed to the upload directory on ftp.franken.de as
described in my package report.
