The Cygwin tools are ports of the popular GNU development tools and utilities for Windows 95, 98, and NT. They function by using the Cygwin library which provides a UNIX-like API on top of the Win32 API. More info...

Note
The latest net releases of Cygwin are 1.1.x, which is also sometimes referred to as v1.1. 1.1.x versions of the Cygwin DLL are newer than previous beta Bxx (i.e. B19, B20, B20.1) versions. However, 1.1.x versions of the DLL are still considered beta quality. The last stable release was v1.0 which was only released on CD-ROM. The next stable release will be v1.2 which should be available early next year. To install the latest net release, click on the link above.

Note that the 1.1.x version numbering refers only to the Cygwin DLL. Individual packages like bash, gcc, less, etc. are released independently of the DLL. The setup.exe utility, which is downloaded when you click on the above link, tracks the versions of all installed components.
