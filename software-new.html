JTAG Tools 0.1

JTAG Tools is a software package which enables working with JTAG-aware (IEEE 1149.1) hardware devices (parts) and boards through JTAG adapter. This is first public release. This package has open and modular architecture with ability to writing miscellanous extensions (like board testers, flash memory programmers, and so on). For complete list of supported operating systems, hardware configurations and so on see the homepage. Nov 25 2002, Marcel Telka
