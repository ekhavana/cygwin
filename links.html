Mingw:

Mingw is a gcc compiler configuration that does without the Cygwin Unix emulation layer. Instead, it uses the Microsoft CRTDLL as its C library. If you want a Win32 native compiler without Unix support, this is for you! The compiler in B20 includes a new -mno-cygwin flag that pulls in the Mingw libraries instead of Cygwin. In some future release, Mingw may become the default with a -mcygwin flag available to optionally include the Cygwin library. This would be the new ix86-win32 configuration of gcc.

  • The original Mingw site by Colin Peters who started Mingw. The version of Mingw included in B20 is based on Colin Peters' July 1, 1998 Mingw snapshot plus some additional patches.
  • Jan Jaap's site containing gcc 2.8 binaries for Mingw and other related resources
  • Mumit Khan's i386-mingw32 native toolchain based on egcs is available here.
