Chapter 1. Cygwin/XFree Overview

What is it?

Cygwin/XFree is a project aiming to port XFree86 to the Win32 platform. Cygwin/XFree is actively being developed, so you should expect to encounter problems. At the time of writing, supported Win32 platforms are Windows Me, Windows NT 4.0 SP4+, and Windows 2000; Windows 95 and Windows 98 are not supported. Please consult the Cygwin/XFree FAQ at http://sources.redhat.com/cygwin/xfree/faq.html for the most recent list of supported platforms.
