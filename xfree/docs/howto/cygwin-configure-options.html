Command Line Parameters

You need to pass a proper screen configuration for screen number 0 to XWin.exe; below is the syntax for XWin.exe parameters.

XWin.exe [-screen screen_number widthxheightxbit_depth...]
[-blackpixel pixel_value] [-whitepixel pixel_value]

Below is an example command line for XWin.exe.

XWin.exe -screen 0 800x600x8 -whitepixel 255 -blackpixel 0
