Chapter 1. Cygwin/XFree86 Overview

What is it?

Cygwin/XFree86 is a project aiming to port XFree86 to the Win32 platform. Cygwin/XFree86 is actively being developed, so you should expect to encounter problems. At the time of writing, supported Win32 platforms are Windows Me, Windows NT 4.0 SP4+, and Windows 2000; Windows 95 and Windows 98 are not supported. Please consult the Cygwin/XFree86 FAQ at http://xfree86.cygwin.com/faq.html for the most recent list of supported platforms.
