Shared Memory Support (MIT-SHM Extension)

Shared memory is provided by the cygserver service. On Windows NT, Windows 2000 and Windows XP cygserver can be installed as a windows service.

To install cygserver run the cygserver-config script. It will install the service.

XWin.exe will not use Shared Memory if the CYGWIN does not contain the server keyword. Starting XWin.exe with 
CYGWIN=server XWin.exe
will enable Shared Memory support if the cygserver service is running.
