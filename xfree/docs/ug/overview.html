Chapter 1. Cygwin/X Overview

Cygwin/X is a port of the X Window System to Cygwin; Cygwin provides a UNIX-like API on the Win32 platform. As of 2004-03-09 the supported Win32 platforms are Windows 95, Windows 98, Windows Me, Windows NT 4.0, Windows 2000, Windows XP, and Windows Server 2003.
