Chapter 1. Cygwin/XFree86 Overview

Cygwin/XFree86 is a port of XFree86 to Cygwin; Cygwin provides a UNIX-like API on the Win32 platform. As of 2001-07-01 the supported Win32 platforms are Windows 95, Windows 98, Windows Me, Windows NT 4.0, and Windows 2000.
