Chapter 1. Cygwin/X Overview

Cygwin/X is a port of XFree86 to Cygwin; Cygwin provides a UNIX-like API on the Win32 platform. As of 2003-01-01 the supported Win32 platforms are Windows 95, Windows 98, Windows Me, Windows NT 4.0, Windows 2000, and Windows XP.
