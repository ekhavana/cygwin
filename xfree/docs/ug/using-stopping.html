Stopping Cygwin/X

When you are done using Cygwin/X, press Alt-F4 to shutdown the X Server; this is the default key combination to shutdown the X Server. The -[no]winkill and -[no]unixkill parameters can change the set of key combinations that will shutdown the X Server. Using the -unixkill allows the Ctrl-Alt-Backspace key combination to shutdown the X Server, in addition to the default Alt-F4 key combination.
