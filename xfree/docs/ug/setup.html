Cygwin/X Contents

Cygwin/X consists of two parts: Cygwin, a port of the GNU tools to Win32, and Cygwin/X, a port of XFree86 to Win32. Both parts of Cygwin/X are needed to successfully run an X Server on a Win32 platform.
