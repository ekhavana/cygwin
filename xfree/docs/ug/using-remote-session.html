Remote Sessions via XDMCP

Cygwin/XFree86 can be used login to a remote session using XDMCP (X Display Manager Control Protocol).

Start Cygwin/XFree86 with a command similar to the following to login to a remote session using XDMCP:

XWin.exe -query remote_host_ip_address -from local_host_ip_address

XDMCP is a complex system; further discussion of XDMCP is beyond the scope of this document. You may wish to read the Linux XDMCP HOWTO for more information.
