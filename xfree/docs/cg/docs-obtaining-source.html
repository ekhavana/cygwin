Obtaining the Source Code

Cygwin/X documentation source code is distributed in a tarred and gzipped archive, seperately from the Cygwin/X X Server source code. The documentation source code is not, as of 2002-01-06, distributed as part of any CVS tree. Updates to the documentation source code are periodically released via the sources.redhat.com mirror network.

Follow these steps to checkout the Cygwin/X documentation source code:

  1. Find your closest sources.redhat.com mirror from the mirror list.

  2. Change to the xfree/devel/documentation/ directory.

  3. Download the latest version of the documentation source code archive, for example, cygwin-xfree-doc-0.9.6.tar.gz (272 KiB).

  4. Unpack the documentation source code archive, for example:

    Username@CygwinHost ~
$ tar -xzf cygwin-xfree-doc-0.9.6.tar.gz

  5. You should now have a directory with the base name of the archive that contains the Cygwin/X documentation source code. For example:

    Username@CygwinHost ~
$ ls
cygwin-xfree-doc-0.9.6  cygwin-xfree-doc-0.9.6.tar.gz
