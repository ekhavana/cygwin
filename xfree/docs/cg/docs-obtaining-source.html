Obtaining the Source Code

As of 2004/04/10 this section of the manual needs to be rewritten.

The documentation source code is available from freedesktop.org cvs. The obtain them please follow these rules: 
$ cvs -d :pserver:anoncvs@pdx.freedesktop.org:/cvs/xserver login
CVS password: <hit return>
$ cvs -d :pserver:anoncvs@pdx.freedesktop.org:/cvs/xserver co cygwin-x-doc
You should now have the sources in an directory called cygwin-x-doc.
