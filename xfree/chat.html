Cygwin/X - Chat (IRC)

Last updated: 2004-04-12 1345 EDT

Overview

IRC chat is very useful for diagnosing problems with the help of Cygwin/X developers. IRC chat is also very useful for Cygwin/X developers working on features or coaching newer developers on how to implement a new feature. IRC provides us with an alternative to email that is better suited to some situations in which rapid, short communications are needed (bug fixing, development, etc.). If you have an IRC chat client, please come visit us at irc.freenode.net/#cygwinx!

Please note that though there will always be people listed in the channel they may be away or sleeping or have work to do. If you have a question, then ask. But if you don't get an answer, try the mailing list instead.
