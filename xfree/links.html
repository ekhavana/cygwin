motifdeveloper.com - The completely updated Motif on the World Wide Web, MW3.

Kenton Lee's X Window System and Motif WWW Sites

Virtual Network Computing (VNC) - From the VNC home page, ``[VNC] is, in essence, a remote display system which allows you to view a computing 'desktop' environment not only on the machine where it is running, but from anywhere on the Internet and from a wide variety of machine architectures.''

Win32 X11 Project - X Server for Windows compiled with Microsoft Visual C++ or MinGW (Minimalist GNU for Windows). Win32 X11 is a port of the X Window System to Windows, rather than a port to Cygwin.

The XFree86 Project, Inc.

X.org - In the words of the X.org page, "X.Org is the worldwide consortium empowered with the stewardship and collaborative development of the X Window System technology and standards."

XWinX - Logon to your Windows desktop from a remote X Server. Sort of like VNC.
