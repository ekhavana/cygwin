xwinclip - Cygwin/XFree86 clipboard manager - Test03 release

xwinclip-Test03 has been released, featuring UNIX to DOS string conversion and vice versa. Test02 fixed the NULL character being pasted as part of the string data. xwinclip is a clipboard manager that integrates the Cygwin/XFree86 and Microsoft Windows clipboards. 2001-10-23

Bug fixed: negative sized edge?

The bug that caused crashes with the message "Fatal server error: negative sized edge?" has been confirmed as fixed in the Server Test Series - Test42 release. 2001-08-13

KDE on Cygwin first beta release

First beta release of KDE 1.1.2 for Cygwin is now available. See the KDE on Cygwin project page for more information. 2001-07-23

New files on FTP

Server Test Series binary and source releases and Documentation source releases are now available via the sources.redhat.com ftp mirror network. 2001-07-02

New packaging method

Cygxin/XFree86 binaries are now packed with the XFree86 build-bindist script and installed with the XFree86 Xinstall.sh script. An updated User's Guide has been posted, documeting the new installation process. 2001-07-02
