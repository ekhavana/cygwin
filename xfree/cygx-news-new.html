xwinclip is Available as a setup.exe Package

xwinclip can now be installed by selecting the XFree86-xwinclip package in Cygwin's setup.exe. 2003-01-01

Updated Installation Instructions in the User's Guide

The Cygwin/XFree86 User's Guide has been updated to document the new installation procedure using Cygwin's setup.exe. 2003-01-01

Updated Cross Compiling Instructions in the Contributor's Guide

The Cygwin/XFree86 Contributor's Guide has been updated to document the new cross compiling procedure. You no longer need a host.def file for cross compiling; in fact, be sure to inspect your host.def file and remove it if you had downloaded it according to the previous instructions. 2003-01-01

Rootless Mode Now Supports X Shape Extension

MATSUZAKI Kensuke has added support to the rootless mode for non-rectangular windows, such as those created by the X Shape Extension. This new feature is available in Server Test Series - Test68 and in the XFree86-xserv-4.2.0-15 package; the rootless mode is enabled with the ``-rootless'' command-line flag. 2002-10-20

Preliminary Rootless Support

MATSUZAKI Kensuke has written a preliminary version of a ``rootless'' mode for Cygwin/XFree86. The mode is available in Server Test Series - Test66 and in the XFree86-xserv-4.2.0-13 package; the mode is enabled with the ``-rootless'' command-line flag. 2002-10-16

IEC International Standard prefixes for binary file sizes

We are now using the International Electrotechnical Commission (IEC) International Standard prefixes for binary file sizes, according to the NIST Reference on Constants, Units, and Uncertainty. KiB and MiB mean, respectively, binary kilobyes (1024 bytes) and binary megabytes (1024 binary kilobytes or 1048576 bytes). 2001-11-08

xwinclip - Cygwin/XFree86 clipboard manager - Test03 release

xwinclip-Test03 has been released, featuring UNIX to DOS string conversion and vice versa. Test02 fixed the NULL character being pasted as part of the string data. xwinclip is a clipboard manager that integrates the Cygwin/XFree86 and Microsoft Windows clipboards. 2001-10-23
