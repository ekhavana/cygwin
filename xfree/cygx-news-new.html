IEC International Standard prefixes for binary file sizes

We are now using the International Electrotechnical Commission (IEC) International Standard prefixes for binary file sizes, according to the NIST Reference on Constants, Units, and Uncertainty. KiB and MiB mean, respectively, binary kilobyes (1024 bytes) and binary megabytes (1024 binary kilobytes or 1048576 bytes). 2001-11-08

xwinclip - Cygwin/XFree86 clipboard manager - Test03 release

xwinclip-Test03 has been released, featuring UNIX to DOS string conversion and vice versa. Test02 fixed the NULL character being pasted as part of the string data. xwinclip is a clipboard manager that integrates the Cygwin/XFree86 and Microsoft Windows clipboards. 2001-10-23
