Source code now hosted on freedesktop.org's CVS tree

The Cygwin/X source code is now hosted on freedesktop.org's CVS repository; our particular repository is administered in cooperation with X.org. See our CVS page for information on how to access the source code. 2003-12-01

Xt, Xaw shared libraries fixed - no more keyboard focus problems

Fixes have been made to Xt and Xaw that fixup VendorShell stuff using a DllMain function, which is based on the _Dll_InitTerm used by OS/2 for fixing the same problem. The changes to VendorShell fix the problems with Xt and Xaw-based apps not getting the keyboard focus unless the mouse cursor was over that application's window. 2003-11-17

Xt, Xaw, Xmu now shared libraries

Ralf Habacker figured out how to make Xt a shared library. This drops the XFree86-bin package from 10.3 MiB to 4.1 MiB. Outstanding! 2003-10-17

Preferences system added for tray icon menu and custom icons

Earle F. Philhower III added a system to allow the user to specify arbitrary commands for the system tray menu (e.g. program launching shortcuts) as well as custom icons for certain window classes. This is a huge system and is really amazing. Preferences are stored in ~/.XWinrc or in /usr/X11R6/lib/X11/system.XWinrc, which is formatted as in this example .XWinrc file. 2003-08-27

Cygwin/X 4.3.0 is now available via setup.exe

The headline says it all. Just run Cygwin's installer again and the packages will be automatically selected and installed. 2003-08-01

Finally --- Shutdown Confirmation

A system notification area (tray) icon has been added with an Exit option in the right-click menu. Choosing exit presents the user with an Exit confirmation dialog box. This dialog box has also been added to shutdowns via Alt-F4 and Ctrl-Alt-Backspace. Users have been requesting this feature for years now. 2003-04-30

New Donations Links - Kensuke Matsuzaki and Alexander Gottwald

Links to Kensuke Matsuzaki's (Rootless mode, MultiWindow mode, I18N clipboard support) and Alexander Gottwald's (non-U.S. keyboard detection and configuration, major helper in the mailing list) donations pages have been added to the Donations page. 2003-03-14
