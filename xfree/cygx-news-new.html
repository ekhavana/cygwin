Finally --- Shutdown Confirmation

A system notification area (tray) icon has been added with an Exit option in the right-click menu. Choosing exit presents the user with an Exit confirmation dialog box. This dialog box has also been added to shutdowns via Alt-F4 and Ctrl-Alt-Backspace. Users have been requesting this feature for years now. 2003-04-30

Preliminary Integrated Clipboard Support

The functionality of the standalone xwinclip program has been integrated into the XWin.exe X Server executable. This functionality is off by default; it can be turned on with the -clipboard command-line parameter. Be sure NOT to run xwinclip at the same time as using the -clipboard command-line parameter. 2003-01-20

Preliminary Multi-Window Support

Kensuke Matsuzaki has written a preliminary version of a ``multi-window'' mode for Cygwin/XFree86. The mode is available in Server Test Series - Test70 and in the XFree86-xserv-4.2.0-18 package; the mode is enabled with the ``-multiwindow'' command-line flag. 2003-01-12
