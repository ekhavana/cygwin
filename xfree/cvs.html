Cygwin/XFree in CVS

The sources to the core Cygwin/XFree files are available via anonymous CVS from the Xfree86 CVS servers. If you don't know what CVS is, visit the CVS home page or read the CVS Manual. You'll need a version with networking capabilities to access this server.

Some notes that may help with getting and building the source via CVS from a Windows PC:

  • You need to have cygwin on your machine to compile the source. To install Cygwin, follow this link and use the setup button on that page.
  • Windows users may need to make sure the HOME environment variable is set to something meaningful. (for example create a directory c:\users\home and do SET HOME=C:\users\home). (On Cygwin 1.1.4 and newer this should be automatically created when you first run Bash). 
    After this, you shouldn't have to worry about passwords any more, as it will store your password in $HOME/.cvspass (semi-encoded). In other words, don't type "cvs login" again.
  • On the Xfree86.org CVS pages, follow the instructions for "Accessing Anoncvs Via cvs pserver in an UNIX-like environment" Cygwin provides a unix-like environment for windows PC's.
