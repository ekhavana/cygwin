Check out Eric Lassauge's Cygwin/XFree86 Ported Software. Eric has binary versions of many packages available, including xscreensaver, gle, glut, xli, xloadimage, xlockmore, xautolock, gdk-pixbuf, gqview, and GNU bc.

IceWM - ``IceWM is a window manager for the X11 Window System''. Cygwin/XFree86 version.

KDE - KDE Version 1.1.2, Qt 1.45, and Qt 2.3.

LessTif - A source compatible version of OSF/Motif.

OpenDX - From the OpenDX home page,``The Open Source software project based on IBM's Visualization Data Explorer''. README for Cygwin/XFree86.

xfig - ``[A] drawing program for [the] X Window System''. Cygwin/XFree86 version.

xlockmore - Cygwin/XFree86 version.
