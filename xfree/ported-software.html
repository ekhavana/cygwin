Cygwin/X - Ported Software

Last updated: 2003-12-02 0130 EST

Notice

As of 2002-05-12, Cygwin/X is installed via Cygwin's setup.exe. Be sure to check for Cygwin/X packages in setup.exe in the XFree86 category. XFree86 packages that are available via setup.exe will not necessarily be listed here in the future.

Check out Eric Lassauge's Cygwin/X Ported Software. Eric has binary versions of many packages available, including xscreensaver, gle, glut, xli, xloadimage, xlockmore, xautolock, gdk-pixbuf, gqview, and GNU bc.

Gnome - The CygGnome project aims to port Gnome to Cygwin/X.

IceWM - ``IceWM is a window manager for the X11 Window System''. Cygwin/X versions: 1.0.9-2, 1.0.8-6p1.

KDE - The KDE on Cygwin project provides KDE and Qt releases for Cygwin/X.

LessTif - A source compatible version of OSF/Motif. Cygwin/X LessTif binaries are available via setup.exe.

NEdit - From the NEdit home page, ``A multi-purpose text editor for the X Window System, which combines a standard, easy to use, graphical user interface with the thorough functionality and stability required by users who edit text eight hours a day.'' Cygwin/X nedit binares are available via setup.exe.

OpenDX - From the OpenDX home page,``The Open Source software project based on IBM's Visualization Data Explorer''. README for Cygwin/X.

ttmkfdir - From the Joerg Pommnitz's home page,``a small tool that creates a fonts.scale file from a directory with TrueType font files.''. Sam Edge has compiled ttmkfdir for Cygwin/X and he has also included a Cygwin README for ttmkfdir.

Window Maker - Cygwin/X Window Maker binaries are available via setup.exe.

xfig - Cygwin/X xfig binaries are available via setup.exe.

xlockmore - Cygwin/X version.

XTrek - An ``X based multi-player space shoot 'em up game'', according to the man page. Cygwin/X version.
