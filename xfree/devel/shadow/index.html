Last updated: 2001-10-20 1622 EDT

Server Test Series releases are modifications to the Cygwin/XFree86 stable release that are currently being tested. Improvements from the Server Test Series releases are generally incorporated in the XFree86 CVS tree. Contributions to the Cygwin/XFree86 project are more than welcome.

Server Test Series releases run on Windows 95, Windows 98, Windows Me, Windows NT 4.0, Windows 2000, and Windows XP. Color depths of 8, 15, 16, 24, and 32 bits per pixel are supported.

Installing a Server Test Series Release

Server Test is an update to one file, namely XWin.exe, that you install overtop of your current Cygwin/XFree86 installation. Follow the procedures below to backup your original file and install the Server Test file. Run the following commands in a Cygwin shell:

cp XWin-Test50.exe.bz2 /usr/X11R6/bin
cd /usr/X11R6/bin
mv XWin.exe XWin.orig.exe
bunzip2 XWin-Test50.exe.bz2
mv XWin-Test50.exe XWin.exe

Good luck, and enjoy!
