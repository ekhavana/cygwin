Source Code from Cygwin's setup.exe

Use Cygwin's setup.exe to install the xorg-server package, and select the 'src' checkbox to install the source code. Currently this is the X.Org tree with multiple patches applied (including some needed just to build successfully), so this should be the starting point for those wishing to work with the source code.
