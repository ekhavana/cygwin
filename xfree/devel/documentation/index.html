Cygwin/X - Development - Documentation

Last updated: 2003-12-18 2230 EST (Harold L Hunt II)

Cygwin/X documentation aims to make installation and usage of Cygwin/X painless for all users, from the inexperienced to the master code wizard. Screenshots are used when practical to provide users with visual confirmation of the instructions written in the prose.

Cygwin/X documentation is written in XML and is compiled against the DocBook DTD (document type definition).

Patches to the documentation source will be gladly accepted.

Currently all documentation screenshots are in the PNG format; we cannot accept GIF graphics for the documentation. Read the GNU statement regarding GIFs for more information on why we cannot accept GIFs.

Source Code

Cygwin/X documentation sources are distributed with the xserver CVS tree hosted on freedesktop.org.

Use your pdx.freedesktop.org shell account to access /cvs/xserver with ssh, or use anonymous CVS:

$ cvs -d :pserver:pdx.freedesktop.org:/cvs/xserver login 
CVS password: <hit return>
$ cvs -z3 -d :pserver:anoncvs@pdx.freedesktop.org:/cvs/xserver co cygwin-x-doc

If you just want to look at the Cygwin/X documentation source, use the View CVS interface to the Cygwin/X documentation tree.

Documentation Source Code (XML)

As of 2004-03-09, a 'cygwin-x-doc' package was added to Cygwin's setup.exe that contains both a compiled version of the documentation and the source code. This package can be found under the 'Doc' category in setup.exe.
