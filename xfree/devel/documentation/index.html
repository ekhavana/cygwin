Cygwin/XFree86 Documentation Development

Last updated: 25Feb2001 1725 EST

Cygwin/XFree86 documentation aims to make installation and usage of Cygwin/XFree86 painless for all users, from the inexperienced to the master code wizard. Screenshots are used when practical to provide users with visual confirmation of the instructions written in the prose.

Cygwin/XFree86 documentation is written in SGML and is compiled against the DocBook DTD (document type definition). Currently the documentation source is only compiled into HTML form, but the SGML source may be compiled into RTF, PDF, and DVI.

Patches to the documentation source will be gladly accepted.
