Cygwin/XFree86 - Development

Last updated: 2001-07-02 0234 EDT

Current development efforts are focused on a shadow (xc/programs/Xserver/miext/shadow) framebuffer (xc/programs/Xserver/fb) based server that uses three different methods of getting the updated bits to the screen: the GDI blitter, the DirectDraw blitter, and the DirectDraw4 blitter.

A side development area is implementing each of the X graphics calls using native Windows GDI graphics calls; this style of server will not require a shadow framebuffer, nor will it require access to the primary framebuffer. The framework for the native GDI server is contained in the sources for the shadow framebuffer series of servers.

Server Test Series - contains binaries, sources, and change logs for updates to the Cygwin/XFree86 server executable. Contributions to the Cygwin/XFree86 Server Test Series are more than welcome.

Documentation - contains compiled documents, sources, and change logs for the Cygwin/XFree86 documentation. Contributions to the Cygwin/XFree86 Documentation are more than welcome.

Contributing - contains some information on how to start contributing to Cygwin/XFree86, and descriptions of, and links to, helpful documentats and books.
