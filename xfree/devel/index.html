Cygwin/X - Development

Last updated: 2003-12-01 2330 EST

Change Logs - Links to the change logs for the Documentation, XWin Server, and xwinclip.

Contributing - contains information on how to start contributing to Cygwin/X, and descriptions of, and links to, helpful documents and books.

Documentation - contains compiled documents, sources, and change logs for the Cygwin/X documentation. Contributions to the Cygwin/X Documentation are more than welcome.

Donations - if you can not contribute time for development, then perhaps you can contribute a little money or hardware to make the current developers more productive.

XWin Server - sources, and change logs for updates to the Cygwin/X server executable. Contributions to the Cygwin/X XWin Server are more than welcome.

Source in CVS - contains information about where the Cygwin/X source code is located, how to obtain that source code, etc.

xwinclip - contains binaries, sources, and change logs for updates to the clipboard manager that integrates the Cygwin/X and Microsoft Windows clipboards. Contributions xwinclip are more than welcome.
