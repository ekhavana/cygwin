Cygwin/XFree86 Development Home

Last updated: 24Feb2001 2018 EST

Current development efforts are focused on implementing each of the X graphics calls using Windows GDI graphics calls. Check out the GDI Server Development Framework if you are interested in contributing to Cygwin/XFree86 development.

The Offscreen (framebuffer) and No Direct Input development branch has served its purpose and is not being further developed.
