So do I have to buy it or what?

The Cygwin DLL is offered under the same license as many other free software packages -- it uses the GNU GENERAL PUBLIC LICENSE (GPL).

This means that if your program uses the Cygwin DLL, you must also provide source code for your program.

There is an alternative to the above free software method of using the Cygwin DLL. Red Hat also provides a "commercial use" license which allows you to distribute only binary copies of your program and the Cygwin DLL. This method requires the purchase of a license from Red Hat.

Obviously, the license that is appropriate for you is entirely based on your requirements. If you cannot distribute source code, then you need the comercial license. If you are able to distribute source code, then the free software license is probably appropriate.

The above information provides links to information about both licensing methods. If you are interested in purchasing a commercial license, then you will eventually need to communicate with a Red Hat employee. Please be advised however, that the person you will be communicating with will be a sales person. They will not be a lawyer. They will be able to provide you with information on purchasing a commercial license. They will be able to help you decide if you need a commercial license. They will not be able to help you to figure out if you are correctly adhering to the GPL. If you have concerns about your distribution of your source code please contact your lawyer. Unfortunately, we can't offer free legal advice.
