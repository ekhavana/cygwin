Porting UNIX Programs to Win32 with Cygwin

To give you some idea of the power of the Cygwin API, we've put together a list of some of the useful programs that you can compile and run using the Cygwin tools.

These are not officially supported Cygwin packages. If you have problems with any of these packages, please contact the owner of the package for advice. If this is not feasible, please make it clear that you are referring to a contributed package when asking for help in the cygwin mailing list.
